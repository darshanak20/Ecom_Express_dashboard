# Ecom Express Power BI Dashboard

## Overview
This project presents an interactive Power BI dashboard analyzing e-commerce order performance and revenue loss due to cancellations. The dashboard focuses on identifying high-revenue products, cancellation impact, and monthly revenue trends.

## Key Metrics
- Total Revenue
- Average Order Value (AOV)
- Cancellation Rate
- Lost Revenue from Cancelled Orders
- Total Orders

## Dashboard Highlights
- Product-wise analysis of lost revenue
- Monthly revenue trend analysis
- Category-based filtering for focused insights
- Identification of top products contributing to revenue loss

## Tools Used
- Power BI
- DAX
- Microsoft Excel / CSV

## Business Insights
- A small set of products contributes disproportionately to revenue loss.
- Cancellation rate significantly impacts overall revenue.
- Revenue shows seasonal trends with noticeable monthly variations.

## How to Use
- Use category slicer to filter dashboard insights.
- Hover over visuals for detailed tooltips.
- Analyze top products to reduce cancellation-related losses.
